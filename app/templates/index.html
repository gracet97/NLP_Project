About This Project

The problem being addressed is how the language used in a letter influences the chances of the patient attending the subsequent appointment, and how the language could be adjusted using natural language processing to achieve improved attendance.

Within the health service there is a considerable gap between the number of people invited to attend an appointment and those who actually turn up. The difference is referred to as the conversion rate. Conversion rates can often be low.

Kainos operate a product called Evolve, which is used to send out appointment letters to medical patients. An automated program iterates through a database to identify users who require a letter sent to them, then sends this from a mail merge email template. Unfortunately, the conversion rate of this approach is less than 10%.

The solution proposed is a web application that accepts text input in the form of a letter and returns a score for based on the letter’s expected conversion rate, based on the language used. It highlights the keywords and phrases that lead to this conclusion and therefore will allow the user to improve the letter in order to improve the conversion rate.

Natural language processing is used to identify keywords that analyse the letter for the following -

  • Sentiment Analysis
  • Entity Analysis
  • Personality Analysis.
