Services

This product analyses the text of letters. It offers sentiment analysis, entity analysis and personality analysis.
Find out more about each option below.

Sentiment Analysis

AWS Comprehend is used to analyse the sentiment present in the text. It states whether the text is negative, neutral or positive. It provides a score for this rating, in order to show how much the text represents the sentiment returned.

Entity Analysis

AWS Comprehend is used to identify the entities present in the text. It states what type of entity it has identified. It provides a score for this identification, in order to show how confident the system is that the entity has been identified correctly.

Personality Analysis

A text classifier is used to detect the presence of the Big 5 personality types in the text. These personality types are - Openness, Conscientiousness, Extraversion, Agreeableness and Neuroticism. It provides a list of the personalities of these 5 which are present above the average weighting.

About this Project

The problem being addressed is how the language used in a letter influences the chances of the patient attending the subsequent appointment, and how the language could be adjusted using natural language processing to achieve improved attendance.

Within the health service there is a considerable gap between the number of people invited to attend an appointment and those who actually turn up. The difference is referred to as the conversion rate. Conversion rates can often be low.

The solution proposed is a web application that accepts text input in the form of a letter and returns analyse of the letter that could be linked to its expected conversion rate, based on the language used.

Natural language processing is used to complete the following analyse on the letter's text -

  • Sentiment Analysis
  • Entity Analysis
  • Personality Analysis

The personalities assessed within personality analysis are the Big Five personality types, also known as OCEAN, which are as follows -

  • Openness
  • Conscientiousness
  • Agreeableness
  • Extraversion
  • Neuroticism

If you wish to see how you score for each of the Big Five, you can take a free quiz here.

The goal is that this product will allow healthcare professionals to enter the text of a letter, to then edit it in order to make it appeal to the common case. Regarding sentiment analysis, it is assumed that a letter expressing positive sentiment will allow for an increase in conversion rate. Regarding entity analysis, it is assumed that a letter with clear entities that include the patient's name, and the date and location of the appointment, will allow for an increase in conversion rate. Regarding personality analysis, it is assumed that a letter expressing the average level of each personality trait will allow for an increase in conversion rate.

This would lead to many benefits, one of the which being budgetary savings due to the increased efficiency of staffs' time. With this money being used elsewhere, and more people attending appointments, the aim is that public healthcare will increase.
