About this Project

The problem being addressed is how the language used in a letter influences the chances of the patient attending the subsequent appointment, and how the language could be adjusted using natural language processing to achieve improved attendance.

Within the health service there is a considerable gap between the number of people invited to attend an appointment and those who actually turn up. The difference is referred to as the conversion rate. Conversion rates can often be low.

Kainos operate a product called Evolve, which is used to send out appointment letters to medical patients. An automated program iterates through a database to identify users who require a letter sent to them, then sends this from a mail merge email template. Unfortunately, the conversion rate of this approach is less than 10%.

The solution proposed is a web application that accepts text input in the form of a letter and returns analyse of the letter that could be linked to its expected conversion rate, based on the language used.

Natural language processing is used to complete the following analyse on the letter's text -

  • Sentiment Analysis
  • Entity Analysis
  • Personality Analysis

The personalities assessed within personality analysis are the Big Five personality types, also known as OCEAN, which are as follows -

  • Openness
  • Conscientiousness
  • Agreeableness
  • Extraversion
  • Neuroticism

If you wish to see how you score for each of the Big Five, you can take a free quiz here.

The goal is that this product will allow healthcare professionals to edit a letter in order to make it appeal to the common case. This would lead to many benefits, one of the which being budgetary savings due to the increased efficiency of staffs' time. With this money being used elsewhere, and more people attending appointments, the aim is that public healthcare will increase.
